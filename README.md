# Arbitrary

这个仓库用于存放一些随手写的，没有目的，不知道放哪里好的，但是又有一些有趣的代码。

## Pages

一些可运行在浏览器里的东西会被发布到这个 Repo 的 [Github Page][gh-page] 上，如果感兴趣可以过去看看。

## 目录

- `virtualmap`：对二维坐标系上的 BFS 算法的一次可视化尝试。
- `bezier`: 贝塞尔曲线测试工具。
